# marketingos

The Python backend for **MarketingOS** — a ten-agent pipeline (research →
strategy → plan → copy → creative → QA → packaging) that turns one website URL
into a packaged first-week marketing campaign, served over FastAPI.

See the [repository README](../README.md) for the full overview and quickstart,
and [docs/REPRODUCE_README.md](docs/REPRODUCE_README.md) for step-by-step
reproduction instructions.
